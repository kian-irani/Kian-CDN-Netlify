اطلاعات از فروشنده VPS — هیچ‌جا ذخیره نمی‌شن، فقط داخل مرورگرته.

آدرس IPv4 سرور
معمولاً ۲۲
پورت relay: به صورت خودکار پورت 8888 انتخاب شده تا با سرویس‌های دیگه تداخل نداشته باشه.
۱
اتصال به VPS با Termius — رایگان
📲 دانلود Termius
راهنمای اتصال:
۱. Termius → دکمه + → New Host
۲. Hostname: IP سرورت | Port: 22
۳. Username: root | Password: رمز VPS
۴. Save → روی هاست tap کن
۲
اسکریپت نصب خودکار یه بار paste کن — همه کار انجام می‌شه
ابتدا IP سرور را در مرحله ۰ وارد کن
✅ فایروال غیرفعال می‌شه (پورت‌های موجود دست نمی‌خوره)
✅ 3x-ui نصب و تنظیم می‌شه
✅ inbound VLESS+XHTTP روی پورت 8888 ساخته می‌شه
✅ UUID و PATH نمایش داده می‌شه
اسکریپت شخصی‌سازی‌شده — همه رو کپی و paste کن
...
بعد از اجرا: مقادیر UUID=... و PATH=... چاپ می‌شن — اونا رو در مرحله بعد وارد کن.
۳
وارد کردن UUID و PATH از خروجی اسکریپت
خروجی اسکریپت:
UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
PATH=relay-xxxxxxxx
کل خط بعد از UUID= رو کپی کن
بدون / — فقط همون چیزی که بعد از PATH= نوشته
۴
راه‌اندازی Netlify رایگان — بدون کارت بانکی
ابتدا IP سرور را در مرحله ۰ وارد کن
۱. فایل پروژه Netlify رو دانلود کن
این فایل آماده‌ست — فقط دانلود و در Netlify آپلود کن. IP سرورت داخلش جاسازی شده.
۲. آپلود در Netlify:
🌐 باز کردن Netlify Drop
۱. لینک بالا رو باز کن (ثبت‌نام رایگانه)
۲. فایل netlify-relay.zip رو drag & drop کن
۳. صبر کن تا Published ✓ بشه
۴. آدرس xxxx.netlify.app رو کپی کن
اگه خطای Edge Functions داد:
به جای Netlify Drop، از روش GitHub استفاده کن:
• یه اکانت GitHub بساز
• در Netlify → Add new site → Import from Git
• Environment variable اضافه کن:
  Key: TARGET_DOMAIN | Value: http://IP:8888
مثلاً: leafy-banoffee-3a96da.netlify.app
