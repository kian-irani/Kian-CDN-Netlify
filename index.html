آدرس IPv4

فقط برای ساخت اسکریپت استفاده می‌شه — هیچ‌جا ارسال نمی‌شه
۱
اتصال به VPS با Termius — رایگان و ساده
📲 دانلود Termius از Play Store
راهنمای اتصال:
۱. Termius → دکمه + → New Host
۲. Hostname: IP سرورت
۳. Port: 22 | Username: root
۴. Password: رمز VPS → Save → روی هاست tap کن
پیام continue connecting? اومد؟ → Yes بزن — نرماله.
۲
اسکریپت نصب کامل یه بار paste کن — همه کار انجام می‌ده
ابتدا مشخصات VPS را در مرحله ۰ وارد کن

این اسکریپت به صورت خودکار:

✅ فایروال رو غیرفعال می‌کنه (پورت‌های موجود دست نمی‌خوره)
✅ 3x-ui رو نصب و تنظیم می‌کنه
✅ یه inbound VLESS+XHTTP می‌سازه
✅ UUID و PATH رو نمایش می‌ده
🚀 اسکریپت شخصی‌سازی‌شده برای VPS شما
در حال ساخت...
بعد از paste و اجرا:
• نصب ~۲ دقیقه طول می‌کشه
• در آخر UUID=... و PATH=... چاپ می‌شه
• اون مقادیر رو در مرحله بعد وارد کن
۳
وارد کردن UUID و PATH از خروجی اسکریپت

بعد از اجرای اسکریپت، UUID و PATH در ترمینال نشون داده می‌شن. اینجا وارد کن:

خروجی اسکریپت اینطوریه:
UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
PATH=relay-xxxxxxxx
۳۶ کاراکتر — کل خط بعد از UUID= رو کپی کن
بدون / — فقط همون چیزی که بعد از PATH= نوشته
۴
راه‌اندازی Netlify رایگان — بدون کارت بانکی
ابتدا IP سرور را در مرحله ۰ وارد کن
🍴 Fork ریپو در GitHub
بعد از Fork حتماً اسم ریپو رو عوض کن!
Settings → Repository name → یه اسم دیگه (مثلاً my-netlify-relay)
🌐 Import از GitHub در Netlify
مراحل در Netlify:
۱. Add new site → Import an existing project → GitHub
۲. ریپوی fork شده رو انتخاب کن
۳. Build command: npm run build
۴. Publish directory: public
۵. روی Add environment variables کلیک کن:
KEY:
TARGET_DOMAIN
VALUE:
http://IP:8080
۶. Deploy site → صبر کن تا Published ✓
۷. آدرس xxxx.netlify.app رو کپی کن
۵
کانفیگ نهایی VLESS آماده ایمپورت در v2rayNG
ابتدا مراحل ۰، ۳ و ۴ را کامل کن
📱 کانفیگ VLESS نهایی

📱 دانلود v2rayNG (اگه نداری)
ایمپورت در v2rayNG:
۱. دکمه + (بالا راست) → Import config from Clipboard
۲. روی کانفیگ tap کن تا انتخاب بشه
۳. دکمه ▶ پایین رو بزن
۴. یه سایت فیلترشده مثل google.com باز کن
اگه -1ms نشون داد:
• v2rayNG رو آپدیت کن (باید آخرین نسخه باشه)
• یا Hiddify رو امتحان کن
