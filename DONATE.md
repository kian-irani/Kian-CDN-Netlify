- 📡 **Bandwidth** — serving 1,500+ active users
- 🤖 **AI debug bot** — Groq API tokens for free user log analysis
- 🆓 **Free tier for new users** — first 5 invites = free 45-day access

**Your donation directly funds the next month of operation.**
